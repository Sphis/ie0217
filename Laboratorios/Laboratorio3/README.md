# Instrucciones de ejecución
Para compilar el código para este laboratorio se utiliza el siguiente comando:
```
c++ -o test.exe .\principal.cpp .\libro.cpp
```
Seguidamente se corre el ejecutable, que en este caso sería:
```
.\test.exe
```
